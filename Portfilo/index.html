Customer Churn Prediction

Customer Churn Dashboard is a machine learning-powered web application developed using Python, Flask, HTML, CSS, and JavaScript to help businesses analyze customer churn and make data-driven retention decisions. The dashboard enables users to upload customer datasets, visualize churn trends through interactive charts, predict customer churn using a Random Forest machine learning model, calculate revenue at risk, simulate retention strategies, filter data by date, explore customer segments, interact with an AI-powered Churn Assistant chatbot, and generate professional PDF reports. The application provides actionable insights that help organizations identify at-risk customers and improve customer retention strategies.

Handwritten Digit Recognition

Handwritten Digit Recognition is a deep learning-based application that accurately identifies handwritten digits (0–9) from uploaded images using a Convolutional Neural Network (CNN). Developed with Python, TensorFlow, Keras, OpenCV, and Flask, the system preprocesses input images, extracts features, and predicts the corresponding digit with high accuracy. The project demonstrates the practical application of computer vision and artificial intelligence in image classification tasks.
